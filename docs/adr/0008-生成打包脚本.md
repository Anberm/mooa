## 决策

对于 APP 来说，CLI 应支持以下功能：

 - 从 ``package.json`` 中获取 appName
 - 从 ``dist\`` 目录/配置服务器中，获取打包完后的文件，生成 ``apps.json`` 文件
 - 更新 ``main.ts`` 对于 MooaPlatform 的引用
 - 在 ``app.module.ts`` 中更新 ``APP_BASE_HREF``
 - 更新对 ``zone.js`` 文件的引用
 
对于 Host 来说，CLI 拥有自己的配置文件：

 - 包含每个应用的线上地址
 - 包含 app 的基本配置
